# Sub-Agents

> Place sub-agent definitions here (as .md files).
> Sub-agents are specialized Claude personas you can call on for specific tasks.
>
> Example: A file called `copywriter.md` could define a sub-agent that writes marketing copy
> using the FINDGOD brand voice and ICP automatically.
>
> Each file should describe the agent's role, what it has access to, and how it should behave.
