# Custom Slash Commands

> Place custom slash command files here (as .md files).
> These let you create shortcuts you can run in Claude Code with /project:command-name.
>
> Example: A file called `daily-post.md` in this folder becomes the command `/project:daily-post`,
> which could generate the next day's Instagram post in FINDGOD's voice.
>
> Each file should contain the prompt or instructions Claude should follow when the command is run.
